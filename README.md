# FUTURE_CS_01
Vulnerability Assessment Report for http://testphp.vulnweb.com/
This report contains a simplified Vulnerability Assessment report for the website:http://testphp.vulnweb.com/ The website is an intentionally vulnerable environment created for cybersecurity learning and practice.
# Objective-The goal of this report was to:
- Learn penetration testing basics
- Identify security vulnerabilities
- Understand how attacks work
- Practice writing a security report.
# Tools Used:
- Nmap
- OWASP ZAP
- Burp Suite
# Vulnerabilities Found
1. SQL Injection
2. Weak Authentication
3. Cross-Site Scripting (XSS)
4. Directory Indexing
5. PHP-CGI Misconfiguration
